# Data_Structure

This repository contains multiple Java practice projects and learning modules developed in Eclipse.
It includes examples for data structures, algorithms, collections, file handling, object-oriented programming, and JDBC basics.

## Project Overview

The workspace is organized as multiple Java projects:

- `abcd`: Basic Java classes and module setup.
- `Aish`: Simple Java entry-point based project.
- `JavaPractice`: Main learning project with many topic-wise packages.
- `JDBC`: JDBC practice setup and module structure.
- `mypro`: Additional modular Java practice project.

The `navya` folder is intentionally excluded from Git tracking.

## Key Topics Covered

- Arrays and string-based coding problems
- Sorting and pattern problems
- Java Collection Framework
- Exception handling
- File handling and buffered I/O
- Interfaces, lambda expressions, and inner classes
- Object arrays and custom class practice
- JDBC fundamentals

## How To Run

### Option 1: Eclipse IDE

1. Open Eclipse.
2. Import existing projects from `eclipse-workspace`.
3. Choose any class with a `main` method.
4. Run as Java Application.

### Option 2: Command Line (Example)

Compile:

```bash
javac -d bin src/Main.java
```

Run:

```bash
java -cp bin Main
```

Use the correct package-qualified class name for packaged classes.
